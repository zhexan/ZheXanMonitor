## 技术详细介绍

### 后端

- **Spring Boot**: Spring Boot 是 Spring 框架的快速开发脚手架，自动配置简化了 Spring 应用的搭建
- **Spring Security**: 提供身份认证和授权功能，保护 API 端点
- **JWT**: 无状态身份验证机制，用于用户登录和 API 访问控制
- **MyBatis Plus**: MyBatis 的增强工具，简化数据库操作
- **Redis**: 缓存用户会话、Token 等热点数据
- **InfluxDB**: 时序数据库，存储服务器监控指标（CPU、内存、磁盘等）
- **RabbitMQ**: 消息队列，用于异步任务和系统解耦
- **WebSocket**: 客户端与服务端实时通信
- **JSch**: 实现 SSH 远程连接，执行远程命令
- **OSHI**: Java 跨平台系统信息监控库
- **Smile**: 机器学习库，用于数据分析

### 前端

- **Vue 3**: 渐进式前端框架，采用 Composition API
- **Vite**: 下一代前端构建工具，开发体验好
- **Pinia**: Vue 3 推荐的状态管理库
- **Element Plus**: Vue 3 组件库，提供丰富 UI 组件
- **ECharts**: 可视化图表库，展示监控数据
- **xterm.js**: Web 终端实现，支持 SSH 远程连接
